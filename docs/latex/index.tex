Bbop-\/2D is a graphics library based on \href{https://www.opengl.org/}{\texttt{ Open\+GL}}, specifically designed for {\bfseries{2D game development}}. ~\newline
 It stands out with its support for {\bfseries{lighting and normal maps}}, enabling the creation of visually rich and immersive environments, even in a 2D context.
